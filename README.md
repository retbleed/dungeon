# Dungeon


## How to run [MacOS Only]

This will create the build folder
```bash
make prepare
```


This will generate the files to build
```bash
make prebuild
```


Follow this commands to make the build. (Haven't figure it out how to do it from the Makefile)
```bash
cd build
make
cd ..
```

Run the program
```bash
make run
```
