# Dungeon
- [X] Run on MacOS ARM64
- [X] Run on Linux
- [ ] Run on Windows


## How to run [MacOS & Linux Only]
Get gmake
You can run this with `make` also.
```bash
brew install make 
```

This will create the build folder
```bash
gmake prepare
```


This will generate the files to build
```bash
gmake prebuild
```


Follow this commands to make the build.
```bash
gamke to-build
```

Run the program
```bash
gmake run
```
